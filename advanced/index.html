4 Lisätoiminnot – mitä opit tässä osiossa

Tässä osiossa käydään läpi muutamia edistyneempiä GitHubin lisätoimintoja. Opit muun muassa automaatiosta GitHub Actionsin avulla, projektinhallinnasta GitHub Projectsin avulla sekä löydät ohjeita itsenäiseen jatko-opiskeluun aiheesta.

  1. Automaatio (GitHub Actions)
    GitHub Actions -automaatiotyökalun perusteet ja käyttöönotto.
  2. Projektinhallinta (GitHub Projects)
    GitHub Projects -työkalun käyttö projektinhallinnassa.
  3. Jatko-opiskelu
    Linkit Gitin ja GitHubin syvällisempään opiskeluun.
