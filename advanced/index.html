4 Lisätoiminnot – mitä opit tässä osiossa

Tässä osiossa käydään läpi muutamia edistyneempiä GitHubin lisätoimintoja. Opit projektinhallinnasta GitHub Projects:illa, automaatiosta GitHub Actions:in avulla, sekä löydät ohjeita itsenäiseen jatko-opiskeluun Gitin ja GitHubin parissa.

  1. Projektinhallinta (GitHub Projects)
    Projektinhallinta yleisesti, sekä GitHub Projects -työkalun käyttö projektinhallinnassa.
  2. Automaatio (GitHub Actions)
    Automaatio yleisesti, sekä GitHub Actions -automaatiotyökalun perusteet ja käyttöönotto.
  3. Jatko-opiskelu
    Linkit Gitin ja GitHubin lisäresursseihin ja oppaisiin, joiden avulla voit syventää osaamistasi aiheesta.
