Projektitaulun luominen

Tässä harjoituksessa luot oman GitHub Projects -taulun tietylle repositoriolle ja harjoittelet sen avulla projektin tehtävien hallintaa.

Harjoitus 8 - GitHub Projects repositoriokohtaisen projektitaulun luominen
  1. Luo GitHub-repositorio tai hyödynnä olemassa olevaa
  2. Siirry repositorion Projects -välilehdelle
  3. Luo uusi taulu nimeltä Git-harjoitukset
  4. Valitse Team Planning -template
  5. Tarkasta että sen Board-näkymä sisältää sarakkeet To do, In progress, Done
  6. Luo kolme tehtävää To do -sarakkeeseen (esim.Tee branch, Tee muutokset Tee PR)
  7. Määritä itsesi tehtävien vastuuhenkilöksi ja määritä prioriteetti, koko, sekä target date
  8. Liitä yksi tiketti uuteen pull requestiin, jotta sen tila päivittyy automaattisesti kun lisäät pull requestiin tarvittavat tiedot
  9. Lopuksi tarkista että tiketti siirtyi oikeaan sarakkeeseen

Kokeile rohkeasti GitHub Projectsin eri toimintoja ja ominaisuuksia!
