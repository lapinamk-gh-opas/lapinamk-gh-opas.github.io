Projektitaulun luominen

Harjoituksessa luodaan oma GitHub Projects -taulu, jossa hallitaan projektin tehtäviä.

Harjoitus 8 - GitHub Projects projektitaulun luominen
  1. Luo GitHub-repositorio tai hyödynnä olemassa olevaa
  2. Siirry repositorion Projects -välilehdelle
  3. Luo uusi taulu nimeltä Git-harjoitukset
  4. Valitse valmiista mallista "Team Planning"
  5. Tarkasta sarakkeet: To do, In progress, Done
  6. Luo kolme tehtävää (esim. Kloonaa repo, Tee branch, Tee commit ja push)
  7. Määritä itsesi tehtävien vastuuhenkilöksi ja lisää iteraatio ja
  8. Siirrä tikettejä sitä mukaa, kun tehtävät etenevät
  9. Lopuksi liitä yksi tiketti pull requestiin ja tarkastele, miten tila päivittyy automaattisesti

Kokeile rohkeasti GitHub Projects:in eri toimintoja ja ominaisuuksia!
