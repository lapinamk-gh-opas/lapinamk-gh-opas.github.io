GitHub Actions – ensimmäinen workflow

Tässä harjoituksessa otetaan käyttöön yksinkertainen GitHub Actions -työnkulku, joka suoritetaan automaattisesti aina, kun teet muutoksia repositorion päähaaraan ja workflown työ kirjoittaa Actions lokiin tekstin "Hello World!". Tavoitteena on ymmärtää, mistä kohdasta workflowt löytyvät GitHubissa, miten ne käynnistyvät ja miten lopputulosta tulkitaan. Harjoituksen tekemiseen riittää oppaan example.yaml–tiedoston pohjan ymmärtäminen ja repositoriossa työskentelyn perustaidot.

Harjoitus 9 – Ensimmäisen GitHub Actions -workflow’n luominen
  1. Luo repositorioon kansiot .github/workflows
    • Avaa repositorio paikallisesti
    • Luo juureen tarvittavat kansiot
  2. Luo workflow-tiedosto
    • Luo tiedosto nimellä tervehdys-workflow.yml
    • Lisää tiedostoon sisältö jolla:
      • Workflown nimeksi annetaan Harjoitus 9
      • Workflow käynnistyy päähaaraan suoritettavalla push toiminnolla
      • Lisättään työ (jobs), jonka nimi on tervehdys-lokiin
      • Työ ajetaan uusimmalla Linux Ubuntu -käyttöjärjestelmällä
      • Lisää askel, joka suorittaa scriptin echo "Hello world!"
  3. Vie workflow GitHubiin
    • Commitoi ja puske muutokset päähaaraan (git push origin main)
  4. Tarkista GitHubissa että workflow on ajettu onnistuneesti
    • Mene repositoriosi Actions -välilehdelle
    • Varmista, että workflown suoritus onnistui ja näkyy vihreänä
  5. Tarkastele lokia
    • Avaa workflown loki
    • Etsi lokissa rivi: Hello World!
  6. Muokkaa workflowta ja käynnistä se uudelleen
    • Lisää toinen askel, joka suorittaa scriptin esim.: echo "Näin suoritetaan GitHub Actions workflow!"
    • Puske muutos ja tarkista taas Actionsin lokista tulostus
  7. Varmista, että ymmärrät workflown toiminnan
    • Käynnistyykö workflow aina pushin yhteydessä?
    • Näkyvätkö muutokset lokissa?
    • Osaatko muuttaa komentoja ja nähdä eron?

Jos sait workflow'n toimimaan ja näet tulokset lokissa, osaat tehdä GitHub Action -työnkulun perustason asetukset ja tarkistaa sen toiminnan. Hienoa työtä!
