GitHubin perusteet – Opiskelijan opaaseen

Tämän oppaan tavoitteena on tarjota Lapin AMK:n opiskelijoille selkeä ja käytännönläheinen johdatus versionhallintaan Gitin ja GitHubin avulla. Oppaassa käydään läpi peruskäsitteet, asennus- ja konfigurointiohjeet sekä työskentelytavat, joita tarvitset ryhmäprojekteissa ja tulevassa työelämässä.

Mitä opit

  • Git-versionhallinnan keskeiset komennot ja työnkulut
  • GitHubin käyttö projektien jakamiseen ja yhteistyöhön
  • Tiimityöskentelyn parhaat käytännöt – kuten haarat, commit messaget, pull requestit ja koodikatselmointi
  • Lisäominaisuudet kuten GitHub Pages, GitHub projectit sekä automaatiot GitHub Actionsilla.

Kuinka käyttää opasta

Varsinaiseen sisältöön pääset joko navigointivalikon tai sivuvalikon kautta. Voit edetä luvut järjestyksessä alkaen Perusteista, tai hypätä suoraan sinua kiinnostaviin osioihin. Mikäli sinulla ei ole aikaisempaa kokemusta Git:n ja GitHubin käytöstä on suositeltavaa käydä osiot läpi järjestyksessä. Jokaisessa luvussa on vaiheittaiset ohjeet, esimerkkit sekä harjoituksia, joiden avulla pääset kokeilemaan itse.

Suosittelemme asentamaan Git:n koneellesi ja avaamaan oman GitHub-tilin, jotta voit kokeilla komentoja ja ohjeita käytännössä samalla kun seuraat ohjeita eteenpäin. Asennukseen ja tilin luomiseen saat ohjeet osiossa 1.2 Käyttöönotto.

Tavoite

Oppaan lopussa hallitset Gitin ja GitHubin peruskäytön niin, että pystyt hallitsemaan omia projektejasi ja osallistumaan sujuvasti tiimipohjaiseen ohjelmistokehitykseen.
