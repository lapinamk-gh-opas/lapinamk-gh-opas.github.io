GitHubin perusteet – Opiskelijan oppaaseen

Tämän oppaan tavoitteena on tarjota Lapin AMK:n opiskelijoille selkeä ja käytännönläheinen johdatus versionhallintaan Gitin ja GitHubin avulla. Oppaassa käydään läpi peruskäsitteet, asennus- ja konfigurointiohjeet sekä työskentelytavat, joita tarvitset ryhmäprojekteissa ja tulevassa työelämässä.

Opas on suomen kielellä ja Gitin termit on käännetty suomeksi. Virallisia suomennoksia termeille ei ole, joten oppaassa on käytetty eri lähteistä ja työelämästä kerättyjä termejä.

Mitä opit

  • Git-versionhallinnan keskeiset komennot ja työnkulut
  • GitHubin käyttö projektien jakamiseen ja yhteistyöhön
  • Tiimityöskentelyn parhaat käytännöt, kuten haarat, commit messaget, pull requestit ja koodikatselmointi
  • Käyttämään lisäominaisuuksia kuten GitHub Pages, GitHub Projects sekä GitHub Actions

Kuinka käyttää opasta

Varsinaiseen sisältöön pääset joko navigointivalikon tai sivuvalikon kautta. Voit edetä luvut järjestyksessä alkaen Perusteista, tai hypätä suoraan sinua kiinnostaviin osioihin. Mikäli sinulla ei ole aikaisempaa kokemusta Git:n ja GitHubin käytöstä on suositeltavaa käydä osiot läpi järjestyksessä. Jokaisessa luvussa on vaiheittaiset ohjeet, esimerkkejä, sekä harjoituksia, joiden avulla pääset itse kokeilemaan ja harjoittelemaan käytännössä.

Suosittelemme asentamaan Git:n koneellesi ja avaamaan oman GitHub-tilin, jotta voit kokeilla komentoja käytännössä samalla kun seuraat ohjeita eteenpäin. Asennukseen ja tilin luomiseen löydät ohjeet osiossa 1.2 Käyttöönotto.

Tavoite

Oppaan lopussa osaat Gitin ja GitHubin peruskäytön niin, että pystyt hallitsemaan omia projektejasi ja osallistumaan sujuvasti tiimipohjaiseen ohjelmistokehitykseen.

Muista, että oppiminen on prosessi. Älä epäröi palata aiempiin lukuihin tai harjoituksiin kertaamaan asioita tarpeen mukaan. Tärkeintä on, että opit käyttämään työkaluja luontevasti osana omaa ohjelmistokehitystyötäsi.
