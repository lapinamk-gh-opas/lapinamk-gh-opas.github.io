Osallistuminen projektiin

Tässä harjoituksessa pääset hyödyntämään aiemmin oppimiasi GitHubin käyttöön liittyviä taitoja ja kokeilemaan, millaista on osallistua oikeaan avoimen lähdekoodin projektiin.

Harjoitus 7 - Osallistuminen avoimen lähdekoodin projektiin
  1. Etsi GitHubista first-contributions-repositorio. Oikea hakutulos näkyy alla olevassa kuvassa.
  2. Avaa repositorio ja etsi README.md-tiedosto.
  3. Seuraa README.md-tiedostossa olevia ohjeita. Ohjeissa käydään läpi fork → clone → edit → pull request -työnkulku vaihe vaiheelta. Voit tehdä harjoituksen joko suomeksi tai englanniksi.

Muista lukea joka kohta huolellisesti ennen kuin siirryt seuraavaan vaiheeseen! Avoimen lähdekoodin projekteihin osallistuessa on tärkeää toimia huolellisesti ja noudattaa annettuja ohjeita.
