Osallistuminen projektiin

Tässä harjoituksessa pääset hyödyntämään aiemmin oppimiasi GitHub-taitoja ja kokeilemaan, miltä osallistuminen avoimen lähdekoodin projektiin tuntuu.

Harjoitus 7 - Osallistuminen avoimen lähdekoodin projektiin
  1. Etsi GitHubista first-contributions niminen repositorio (oikea hakutulos näkyy alla olevassa kuvassa)
  2. Avaa repositorio ja etsi README.md -tiedosto
  3. Seuraa README.md-tiedostossa olevia ohjeita. Ohjeissa käydään läpi fork → clone → edit → pull request -työnkulku vaihe vaiheelta. Voit tehdä harjoituksen joko suomeksi tai englanniksi.

Muista lukea joka kohta huolellisesti ennen kuin siirryt seuraavaan vaiheeseen! Avoimissa projekteissa ohjeiden noudattaminen on tärkeää, jotta yhteistyö sujuu.
