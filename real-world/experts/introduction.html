Asiantuntijahaastattelut

Tässä osiossa pääset kuulemaan asiantuntijoiden näkemyksiä ja tosielämän esimerkkejä. Haastattelut tarjoavat arvokasta tietoa siitä, miten Gittiä ja GitHubia käytetään työelämässä. Ne myös auttavat ymmärtämään, miten nämä työkalut voivat parantaa tiimityöskentelyä ja projektinhallintaa.

Haastateltavat

Asiantuntija 1 - FrontEnd Developer
  • 10+ vuotta alan työkokemusta
  • Keskisuuri suomalainen yritys
  • Git ja GitHub käytössä päivittäin

Haastattelun teksti on koottu videohaastattelun pohjalta. Keskustelu pohjautui etukäteen laadittuihin kysymyksiin, mutta eteni vapaamuotoisesti eri aihealueiden ympärille. Vastaukset on kirjattu sisällöllisesti vastaavina (ei sanatarkasti) ja koottu teemoittain.

Asiantuntija 2 - Full Stack Developer
  • Arto Kallanto
  • 10+ vuotta alan työkokemusta
  • Keskon sovelluskehitystiimissä Full Stack Developerina, pääpainon ollessa taustajärjestelmissä.
  • Git ja GitHub käytössä päivittäin

Haastattelu käytiin videohaastatteluna ja tallenne on nähtävissä osiosta Asiantuntija 2. Keskustelun runkona käytetty etukäteen laadittuja kysymyksiä. Videot ovat editoitu ja leikattu, mutta sisältävät kaiken haastattelun aikana käsitellyn asiasisällön. Haastattelu on jaettu neljään osaan/videoon, jotta se olisi miellyttävämpi katsoa.

Toivomme, että haastattelut rohkaisevat sinua oppimaan lisää ja ottamaan Gitin ja GitHubin työkalut käyttöön projekteissasi.
