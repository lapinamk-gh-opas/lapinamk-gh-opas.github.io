Haarat

Perustoiminnot on hallussa ja seuraavaksi kerrataan GitHub branchien eli haarojen käyttöä.

Muista: komennot löytyvät sivupalkin muistilistasta. Voit myös palata edellisiin kappaleisiin, jos tarvitset muistinvirkistystä.

Harjoitus 2 – Haarat
  1. Avaa edellisessä kappaleessa käyttämäsi repo VS Codessa.
  2. Luo kehityshaara päähaarasta selkeällä nimellä (esim. feature/tervehdys).

    Vinkki: jos käytät VS Codea, haaran voi luoda ja vaihtaa myös alareunan statuspalkista.

  3. Siirry luomaasi haaraan.
  4. Muokkaa index.html otsikkoon uusi tervehdys ja lisää uusi <p> -kappale. Esimerkiksi: 
    <h1>Moikka, Git!</h1>
        <p>Tämä teksti lisättiin kehityshaarassa.</p>

    Tallenna ja tarkista muutokset komennolla.

  5. Commitoi muutos selkeällä commit-viestillä.
  6. Julkaise haara GitHubiin.
  7. Varmista, että haara ja muutos näkyvät GitHubissa.
  8. Tarkista mitä haaroja projektillasi on.
  9. Vaihda takaisin päähaaraan (main).
  10. Yhdistä kehityshaara päähaaraan komennolla.
  11. Varmista, että yhdistäminen onnistui ja tarkista tilanne komennolla.
  12. Tarkista, että index.html sisältää haaran muutokset.
  13. Työnnä päähaara GitHubiin ja varmista, että muutokset näkyvät myös siellä.
  14. Poista kehityshaara GitHubista haluamallasi tavalla.
  15. Varmista, että haara on poistettu GitHubista.

Alla olevassa laatikossa on oikeat vastaukset. Huom! Käytä tätä vain, jos olet todella jumissa!

git checkout -b feature/tervehdys
git status
git add index.html
git commit -m "lisää tervehdys"
git push -u origin feature/tervehdys
git branch
git checkout main
git merge feature/tervehdys
git status
git push
git branch -d feature/tervehdys

Lisähaaste (valinnainen): Tee toinen haara, muokkaa samaa riviä eri tavalla ja yritä yhdistää – harjoittele rauhassa konfliktin ratkaisua editorissa.

Huom: jos jokin tökkää, tarkista ensin nykytila (Status)ja kurkkaa sivupalkin muistilistasta oikeat komennot. Tarvittaessa voit myös palata edellisiin osioihin kertaamaan.

Kun olet saanut harjoituksen tehtyä, voit olla ylpeä itsestäsi! Olet oppinut tärkeimmät Gitin ja GitHubin perustoiminnot!
