Git ja GitHub eivät ole sama asia.

  • Git = versionhallintatyökalu, jota käytät omalla tietokoneellasi
  • GitHub = pilvipohjainen palvelu, joka hyödyntää Gitiä taustalla

Tällä sivulla saat perustiedot niiden eroista ja keskeisistä käsitteistä.

Versionhallinta

Versionhallinta on järjestelmä, jolla seurataan ja hallitaan tiedostoihin tehtyjä muutoksia. Sen avulla voidaan palauttaa aiempia versioita, tarkastella projektin historiaa ja työskennellä usean ihmisen kanssa samoissa tiedostoissa yhtä aikaa.

Mikä on Git?

Git on suosittu versionhallintajärjestelmä, jonka loi Linuxin kehittäjä Linus Torvalds vuonna 2005. Se on avoimen lähdekoodin projekti ja täysin ilmainen.

Git toimii paikallisesti omalla tietokoneella, eikä sen käyttö vaadi internet-yhteyttä. Se on suunniteltu erityisesti ohjelmistokehitystä varten, mutta sopii myös muihin versiointia vaativiin projekteihin, kuten animointiin, dokumentointiin ja kirjoittamiseen

Git - keskeiset käsitteet

  • Repository (repo) - Kansio, jossa Git seuraa projektiasi ja sen historiaa
  • Clone - Tee kopio etäreposta omalle tietokoneellesi
  • Commit - Tallenna tehdyt muutokset
  • Branch - Luo haara erillisten versioiden tai ominaisuuksien kehittämistä varten
  • Merge - Yhdistä muutokset eri haaroista (branch)
  • Pull - Tuo viimeisimmät muutokset etäreposta omalle tietokoneellesi
  • Push - Vie omat muutoksesi etärepoon

Huom! Suurin osa Gitin toiminnoista tapahtuu omalla tietokoneellasi. Vain Push ja Pull yhdistävät etäpalvelimeen kuten GitHubiin.

Gitin avulla voidaan
  • seurata ja hallita tehtyjä muutoksia
  • nähdä, kuka muutokset on tehnyt
  • tarkastella projektin koko historiaa ja palata tarvittaessa aiempiin versioihin
Kun Gitin rinnalla käytetään GitHubia, voidaan myös
  • työskennellä usean ihmisen kanssa samassa projektissa ilman, että muutokset menevät päällekkäin
  • tehdä yhteistyötä mistä päin maailmaa tahansa

Mikä on GitHub?

GitHub on pilvipohjainen palvelu ja Git-repositorioiden etäpalvelin. Se tarjoaa Git-versionhallinnan lisäksi työkaluja yhteistyöhön, projektinhallintaan ja automaatioon.

GitHub on maailman suurin lähdekoodin julkaisualusta ja se on ollut Microsoftin omistuksessa vuodesta 2018. GitHubin kaltaisia muita palveluita ovat esimerkiksi GitLab ja Bitbucket.
