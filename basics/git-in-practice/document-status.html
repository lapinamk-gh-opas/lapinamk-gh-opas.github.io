Git-tiedoston tilat (VS Code -näkymä)

Versionhallinta on järjestelmä, jolla seurataan ja hallitaan tiedostoihin tehtyjä muutoksia. Sen avulla voidaan palauttaa aiempia versioita, tarkastella projektin historiaa ja työskennellä usean ihmisen kanssa samoissa tiedostoissa yhtä aikaa.

Tila Väri Selitys
Untracked (U) Vihreä Uusi tiedosto, jota Git ei vielä seuraa. Lisää versiohistoriaan komennolla git add.
Unmodified (-) Ei väriä Ei muutoksia edellisen commitin jälkeen. Synkassa repositorion kanssa.
Modified (M) Keltainen Tiedostoa on muokattu, mutta muutoksia ei ole lisätty staging-alueelle.
Added (A) Vihreä Muutokset on lisätty stagingiin komennolla git add ja ovat valmiina commitille.
Deleted (D) Yliviivattu Tiedosto on poistettu.
Committed (-) Ei väriä Muutokset on tallennettu repositorioon (paikallisesti). Pushaa GitHubiin komennolla git push.
