1. Perusteet – mitä opit tässä osiossa

Tässä osiossa käydään läpi Gitin ja GitHubin käyttöön tarvittavat perusasiat askel askeleelta:

  1. GitHubin idea
    Ymmärrät, mihin Git ja GitHub on tarkoitettu: versionhallinta, yhteiskehitys ja koodin varmuuskopiointi.
  2. Käyttöönotto
    Ohjeet Gitin asentamiseen ja GitHub-tilin luomiseen.
  3. IDE:t ja käyttötavat
    Tutustut eri tapoihin käyttää Gitiä: komentorivi (Git Bash, Terminal), koodieditorit (VS Code, PyCharm, Android Studio) sekä erilliset graafiset sovellukset kuten GitHub Desktop.
  4. Toimintoketju käytännössä
    Harjoittelet koko perustyönkulun: uuden repositorion luonti, tiedostojen lisääminen, commitit ja muutosten lähetys etärepositorioon.
  5. Peruskomennot
    Lyhyt hakemisto tärkeimmistä Git-komennoista ja niiden tarkoituksesta.
  6. Harjoitukset
    Käytännön tehtäviä, joilla voit vahvistaa opittuja taitoja ja testata komentojen hallintaa.
