Gitin konfigurointi

  1. Avaa Git Bash tai terminaali
    • Git Bash asennetaan Windowsille oletuksena Gitin asennuksen yhteydessä
    • Avaa Git Bash Windowsin aloitusvalikosta tai etsi Git Bash
  2. Tarkasta .gitconfig tiedoston sisältö
    • Anna komento git config --global --list

    Tämän tiedoston asetuksia käytetään laitteella globaalisti, eli kaikissa Git-repositorioissa. Asetuksia on mahdollista muuttaa tarvittaessa myös repositorio kohtaisesti. Tämä tapahtuu siirtymällä kansioon, jossa repositorio on ja antamalla komennot ilman --global lippua.

    Jos et ole vielä muokannut asetustiedostoa, se sisältää vain asennuksen yhteydessä määritetyn oletustekstieditorin ( core.editor ). Tässä tapauksessa oletuseditoriksi on asetettu Visual Studio Code, joka valittiin Gitin asennuksen aikana.

  3. Lisää tiedostoon käyttäjänimi ja sähköposti
    • Anna sähköposti komennolla git config --global user.email "sinun@sähköposti.com"
    • Anna nimi komennolla git config --global user.name "Sinun Nimi"
    • Tarkasta muutokset komennolla git config --global --list

    Git käyttää nimeä ja sähköpostia commitin ns. tekijätietoina. Mikäli käytät GitHub-tilillä olevaa sähköpostiosoitetta, osaa GitHub yhdistää commitin sinun tiliisi. Tätä tiedostoa ei kuitenkaan käytetä GitHubissa tunnistautumiseen.

  4. Aseta oletushaaran nimi (main)
    • Aseta nimi komennolla git config --global init.defaultBranch main
    • Tarkasta muutokset komennolla git config --global --list

    Tämä tehtiin jo asennuksen yhteydessä system tasolle, mutta nimeäminen kannattaa varmistaa vielä käyttäjätasolla.

Näillä asetuksilla pääsee hyvin alkuun. Lisää konfiguraatiosta voit katsoa Gitin dokumentaatiosta komennolla git help config

Gitin dokumentaatio voi vaikuttaa aluksi kryptiseltä, joten tässä vielä taulukko jo käytetyistä ja muutamista muista hyödyllisistä asetuksista.

Asetus Kuvaus
user.name Commitin tekijän nimi
user.email Commitin tekijän sähköposti
core.editor Oletuseditorin asettaminen
init.defaultBranch Päähaaran nimeäminen
core.autocrlf Rivinvaihtojen käsittely Windows/Linux
color.ui Värit Git-komentojen tulosteessa
alias.* Lyhenteiden (alias) luominen usein käytetyille komennoille
(esim. git config --global alias.ci commit luo komennolle commit lyhenteen ci )
