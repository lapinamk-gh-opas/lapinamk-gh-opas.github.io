Gitin asennus Linuxille tai macOS:lle

  1. Mene osoitteeseen: https://git-scm.com/downloads
  2. Valitse käyttöjärjestelmä.
  3. Aukeaa sivu, jossa on ohjeet Gitin asentamiseen. Sivulla näkyvät myös paketinhallintakomennot terminaalia varten.
  4. macOS (Homebrew) – asennus Homebrew-paketinhallinnan avulla
    • Avaa Terminal: ⌘ + välilyönti → kirjoita Terminal → Enter
    • Tarkista onko Git jo asennettu:
      • git --version → Enter
      • Esimerkki tulosteesta: git version 2.41.0 (versio voi vaihdella)
    • Asenna Git, jos sitä ei ole tai haluat päivittää:
      • brew install git → Enter
      • Tarkista, että asennus onnistui: git --version → Enter
      • Esimerkki tulosteesta: git version 2.51.0
  5. Linux (Debian/Ubuntu) – asennus pakettivaraston avulla
    • Avaa Terminal
    • Tarkista onko Git jo asennettu:
      • git --version → Enter
      • Esimerkki tulosteesta: git version 2.41.0 (versio voi vaihdella)
    • Asenna Git, jos sitä ei ole tai haluat päivittää:
      • Päivitä pakettivarastot: sudo apt-get update → Enter (Päivittää listan uusimmista saatavilla olevista paketeista, ei itse Git-ohjelmaa)
      • Asenna Git: sudo apt-get install git → Enter
      • Tarkista, että asennus onnistui: git --version → Enter
      • Esimerkki tulosteesta: git version 2.51.0
