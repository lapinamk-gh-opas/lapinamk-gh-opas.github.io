Gitin asennus Linuxille ja macOS:lle

  1. Siirry osoitteeseen https://git-scm.com/downloads
  2. Valitse käyttöjärjestelmä
  3. Seuraavalla sivulla näkyvät käyttöjärjestelmäkohtaiset asennusohjeet sekä paketinhallintakomennot terminaalia varten.
  4. macOS (Homebrew) – asennus Homebrew-paketinhallinnan avulla
    • Avaa Terminal ⌘ + välilyönti → kirjoita Terminal → Enter
    • Tarkista onko Git jo asennettu
      • git --version → Enter
      • Esimerkki tulosteesta git version 2.41.0 (versio voi vaihdella)
    • Asenna Git, jos sitä ei ole tai jos se kaipaa päivitystä
      • brew install git → Enter
      • Tarkista että asennus onnistui git --version → Enter
      • Esimerkki tulosteesta git version 2.51.0
  5. Linux (Debian/Ubuntu) – asennus paketinhallinnan avulla
    • Avaa Terminal
    • Tarkista onko Git jo asennettu
      • git --version → Enter
      • Esimerkki tulosteesta git version 2.41.0 (versio voi vaihdella)
    • Asenna Git, jos sitä ei ole tai jos se kaipaa päivitystä
      • Päivitä pakettilista sudo apt-get update → Enter (Päivittää tiedot uusimmista paketeista, ei vielä asenna Gittiä)
      • Asenna Git sudo apt-get install git → Enter
      • Tarkista että asennus onnistui git --version → Enter
      • Esimerkki tulosteesta git version 2.51.0
