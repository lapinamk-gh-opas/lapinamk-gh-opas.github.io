GitHub-tilit

  • Tilejä on kolmen tyyppisiä:
    • Käyttäjä (User)
    • Organisaatio (Organization)
    • Yritys (Enterprise)
  • Käyttäjätili:
    • Ilmainen versio, joka sopii yksityishenkilöille.
    • Maksullinen Pro-versio tarjoaa käyttäjälle lisäominaisuuksia. (Versio on opiskelijoille ilmainen – varauksin).
    • Personal-tiliä ylläpidät itse.
    • Managed-tiliä ylläpitää yritys tai organisaatio.
  • Organisaatiotili:
    • Ilmainen versio – peruskäyttöön.
    • Maksullinen Team-versio – tarjoaa lisäominaisuuksia tiimeille.
  • Yritystili:
    • Aina maksullinen – suunnattu suurille organisaatioille ja yrityksille.
