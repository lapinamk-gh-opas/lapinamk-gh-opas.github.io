GitHub-tilin luominen

  1. Mene osoitteeseen: https://github.com/
  2. Klikkaa Sign up. Tai vaihtoehtoisesti syötä rekisteröitymiseen käytettävä sähköposti ja klikkaa Sign up for GitHub.
  3. Aukeaa lomake, johon syötät loput tarvittavat tiedot. Klikkaa lopuksi Create account.
  4. GitHub voi pyytää varmistamaan, että olet ihminen (CAPTCHA-varmennus).
  5. Vahvista sähköpostisi. Syötä GitHubin lähettämä vahvistuskoodi ja klikkaa Continue.
  6. Kirjaudu sisään tilillesi. Syötä sähköpostiosoitteesi tai käyttäjänimesi sekä salasana ja klikkaa Sign in.

GitHub vaatii pakollisena kaksivaiheista tunnistautumista (2FA) vain osalle käyttäjistä. Näitä ovat käyttäjät, jotka ovat esimerkiksi: tehneet julkaisun (release) omassa repositoriossa, osallistuneet tärkeisiin avoimen lähdekoodin projekteihin tai hallinnoivat tärkeitä repositorioita. Eli tilin luomisen yhteydessä kaksivaiheinen tunnistautuminen ei ole pakollinen, mutta tietoturvan kannalta erittäin suositeltavaa. Lisäksi GitHub voi muuttaa näitä tunnistautumisen kriteerejä jatkossa parantaakseen tiliturvallisuutta.

  1. Aseta kaksivaiheinen tunnistautuminen käyttöön.
    • Klikkaa oikealta ylhäältä profiilikuvaketta ja valitse sen alta settings
    • Tämän jälkeen valitse vasemmalta sivuvalikosta Password and authentication
    • Paina nappia Enable two-factor authentication
  2. Syötä mobiililaitteen tunnistautumissovelluksen antama avain (esim Microsoft Authenticator tai Google Authenticator).
    • Lisää GitHub tunnistautumissovellukseen lukemalla sovelluksella QR-koodi.
    • Tämän jälkeen syötä sovelluksen antama avain kenttään Verify the code from app.
    • Paina nappia Continue
  3. Ota talteen GitHubin palautus avaimet siltä varalta, että laitteesi häviää tms.
    • Lataa .txt tiedosto palautusavaimista painamalla nappia Download.
    • Tallenna tiedosto turvalliseen ja varmaan paikkaan.
    • Paina nappia I have saved my recovery codes
  4. Kaksivaiheinen tunnistautuminen on otettu käyttöön.
    • Lisää halutessasi muita tunnistautumistapoja
    • Tai klikkaa nappia Done.
