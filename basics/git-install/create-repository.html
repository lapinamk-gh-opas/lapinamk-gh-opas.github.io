GitHub-repositorion luominen

GitHub repositorioiden luominen onnistuu myös komentokehotteella asentamalla GitHub CLI:n, mutta tässä oppaassa GitHubin repositoriot luodaan verkkosivujen kautta selaimella.

  1. GitHub-tilille sisäänkirjautumisen jälkeen on mahdollista mm. luoda etärepositorioita.
    • Klikkaa vasemmalta painikkeesta Create repository.
    • Tai -painikkeella aukeavasta alasvetovalikosta New repository.
  2. Aukeaa lomake, jolla annetaan repositoriolle nimi ja asetuksia.
    • Anna repositoriolle nimi
    • Valitse näkyvyys: Private tai Public
    • Jätä Add README nyt pois päältä (luodaan se itse myöhemmin paikallisesti ja siirretään se repositorioon.)
    • Klikkaa Create repository
    • README.md Kuvaustiedosto, joka kertoo projektin tarkoituksen, käytön ja ohjeet. Näkyy automaattisesti GitHub-repositorion etusivulla.
    • LICENSE Määrittää, miten muut voivat käyttää, muokata ja jakaa julkisesti saatavilla olevaa projektiasi. Esim. MIT- tai GPL-lisenssi.
    • .gitignore Luettelo tiedostoista ja kansioista, joita Git ei seuraa. Käytetään esim. väliaikaisten tiedostojen tai salaisuuksien jättämiseen pois repositoriosta.
  3. Repositorio on nyt luotu. Tyhjässä repositoriossa näkyy keskellä ruutua URL-osoite, jota käyttämällä paikallinen repositorio ja etärepositorio voidaan yhdistää toisiinsa. Tätä siis tarvitaan jatkossa.
