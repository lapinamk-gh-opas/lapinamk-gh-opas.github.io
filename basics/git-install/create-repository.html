GitHub-repositorion luominen

Repositorion luominen on tärkeä osa versionhallintaa ja yhteistyötä. Kuten monissa muissakin Gitin ja GitHubin toiminnoissa, myös repositorion voi luoda usealla eri tavalla.

Yleisimmät tavat repositorion luomiseen
  • GitHubin käyttöliittymässä selaimessa
  • Komentorivillä GitHub CLI-työkalulla
  • GitHub Desktop -sovelluksella

Tässä oppaassa repositoriot luodaan GitHubin käyttöliittymässä selaimessa.

Repositorion luominen GitHubin käyttöliittymästä
  1. Kirjaudu sisään GitHub-tilillesi ja siirry etusivulle
    • Klikkaa vasemmalta painikkeesta Create repository
    • Tai -painikkeella aukeavasta alasvetovalikosta New repository
  2. Täytä repositorion tiedot lomakkeeseen
    • Anna repositoriolle nimi
    • Valitse näkyvyys: Private tai Public
    • Jätä Add README pois päältä (luomme sen itse myöhemmin)
    • Klikkaa Create repository
    • README.md Kuvaustiedosto, joka kertoo projektin tarkoituksen, käytön ja ohjeet. Näkyy automaattisesti GitHub-repositorion etusivulla.
    • LICENSE Määrittää, miten muut voivat käyttää, muokata ja jakaa julkisesti saatavilla olevaa projektiasi. Esim. MIT- tai GPL-lisenssi.
    • .gitignore Luettelo tiedostoista ja kansioista, joita Git ei seuraa. Käytetään esim. väliaikaisten tiedostojen tai salaisuuksien jättämiseen pois repositoriosta.
  3. Repositorio on nyt luotu. Tyhjässä repositoriossa näkyy keskellä ruutua URL-osoite, jonka avulla paikallinen repositorio ja etärepositorio voidaan yhdistää toisiinsa. Tätä tulet tarvitsemaan jatkossa.
