GitHub-repositorion luominen

GitHub repositorioiden luominen onnistuu myös komentokehotteella asentamalla GitHub CLI:n, mutta tässä oppaassa GitHubin repositoriot luodaan verkkosivujen kautta selaimella.

  1. GitHub-tilille sisään kirjautumisen jälkeen on mahdollista mm. luoda etärepositorioita.
    • Klikkaa vasemmalta painikkeesta Create repository.
    • Tai -painikkeella aukeavasta alasvetovalikosta New repository.
  2. Aukeaa lomake, jolla annetaan repositoriolle nimi ja asetuksia.
    • Anna repositoriolle nimi
    • Valitse näkyvyys: Private tai Public
    • Jätä Add README nyt pois päältä (luodaan se itse myöhemmin paikallisesti ja siirretään se repositorioon.)
    • Klikkaa Create repository
  3. Repositorio on nyt luotu. Tyhjässä repositoriossa näkyy keskellä ruutua URL-osoite jota käyttämällä voidaan paikallinen repositorio ja etärepositorio yhdistää toisiinsa. Tätä siis tarvitaan jatkossa.
  4. Siinä vaiheessa, kun GitHub repositorio sisältää tiedostoja löytyy URL-osoite klikkaamalla Code painiketta.
