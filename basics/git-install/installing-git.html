Gitin asennus Windowsille

  1. Siirry osoitteeseen https://git-scm.com/downloads ja valitse Windows-käyttöjärjestelmä.
  2. Lataa asennuspaketti ja käynnistä asennus. Seuraa näytön ohjeita.
  3. Voit käyttää oletusasetuksia, mutta on suositeltavaa vaihtaa uuden repositorion päähaaran nimeksi main . GitHub käyttää tätä oletuksena, joten se helpottaa jatkossa työtäsi. On myös hyvä varmistaa, että Git Credentials Manager on valittu ja tulee asennettua samalla. Tämä helpottaa jatkossa työskentelyä GitHubin kanssa.
  4. Testaa asennus
    • Paina Win + R , kirjoita cmd ja paina OK
    • Kirjoita komentokehotteeseen git --version ja paina Enter
    • Jos näet esimerkiksi git version 2.51.0 (versio voi vaihdella), asennus onnistui
