Gitin käyttö Visual Studio Codella

Tällä sivulla käsitellään Gitin käyttöä Microsoftin kehittämällä koodieditorilla Visual Studio Codella (tai lyhyemmin VS Codella). Haitari elementistä voit katsoa tarvittaessa ohjeet GitHubin käyttöönottoon editorille. Muutoin osiossa luodaan edellisen sivun malliin editorin avulla jälleen tiedosto ja repositorio sekä viedään se sammaan, edelisessä osiossakin käytettyyn, etärepositorioon, mutta käytetään tälle osiolle omaa haaraansa. GitHubin yhdistämisen jälkeen Gitin käytölle VS Coden terminaalista voi hyödyntää edellisen sivun ohjeita.

Avaa terminaali yläpalkista View + Terminal. Git Bashin saa käyttöön terminaaliin -painikkeen vierestä avattavasta alasvetovalikosta.
