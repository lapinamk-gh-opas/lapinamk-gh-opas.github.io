Omat repositoriot löytyvät GitHubista monella eri tavalla. Tässä on niistä yksi.

  • Klikkaa oikealta ylhäältä profiilikuvaketta ja valitse sen alta Repositories
  • Avautuu lista kaikista käyttäjän repositorioista.
  • Repositoriota pääsee tarkastelemaan klikkaamalla sen nimeä.
Repositorion päänäkymästä löydät mm.
  1. Repositoriossa olevat tiedostot ja kansiot
  2. Repositorion Git-haarojen (branch) valikon
  3. Code valikon, jonka alta löydät repositorion avaamiseen tarvittavat tiedot
  • URL-osoitteen näet alasvetovalikosta klikkaamalla Code -painiketta. Voit kopioida URL-osoitteen leikepöydälle klikkaamalla kopioi-kuvaketta.
