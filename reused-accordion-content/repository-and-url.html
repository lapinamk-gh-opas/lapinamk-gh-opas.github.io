• Repositoriot löytyvät useammasta paikkasta, mutta niitä voi selata esimerkiksi näin.
  • Klikkaa oikealta ylhäältä profiilikuvaketta ja valitse sen alta Repositories
  • Avautuu lista kaikista käyttäjän repositorioista.
  • Repositoriota pääsee tarkastelemaan klikkaamalla sen nimeä.
Repositorion päänäkymästä löydät mm.
  1. Repositoriossa olevat tiedostot ja kansiot.
  2. Repositorion Git-haarat valikko.
  3. Code valikon alta löydät mm. repositorion URL-osoitteen.
  • URL-osoitteen näet alasvetovalikosta klikkaamalla Code painiketta.
