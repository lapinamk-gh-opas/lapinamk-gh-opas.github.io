Muutosten tallentaminen etärepoon
git status Tarkista tilanne
git add . tai git add <tiedosto> Lisää muutokset stagingiin
git commit -m "viesti" Tallenna muutokset (commit)
git push Puske muutokset etärepoon
Uuden sivuhaaran luonti
git checkout main Siirry päähaaraan
git pull Hae tuoreimmat muutokset
git checkout -b dev Luo ja siirry haaraan dev
git push -u origin dev Puske haara ja aseta seuranta
Yhdistä muutokset päähaaraan
git checkout main Siirry päähaaraan main
git merge dev Yhdistä sivuhaara dev päähaaraan
Päivitä sivuhaara päähaarasta (main ➜ dev)
git checkout dev Siirry sivuhaaraan dev
git fetch origin Hae tuoreimmat muutokset päähaarasta (origin/main)
git merge origin/main Yhdistä päähaaran muutokset sivuhaaraan
git push origin dev Puske päivitetty dev etärepoon
