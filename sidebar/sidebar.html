  • Komennot muistilista
    Komento Selitys
    cd Siirry kansioon
    cd .. Siirry yksi taso ylemmäs
    dir Näytä kansion sisältö
    mkdir uusiKansio Luo uusi kansio
    git init Alusta uusi Git-repositorio
    git clone <repo> Kloonaa olemassa oleva repositorio
    git remote add origin <url> Yhdistä paikallinen repo etärepositorioon
    git status Näytä muutosten tila
    git add <tiedosto> Lisää tiedosto stagingiin
    git add . Lisää kaikki muutokset stagingiin
    git commit -m "viesti" Tallenna muutokset
    git checkout -b dev Luo ja siirry haaraan dev
    git checkout main Siirry olemassa olevaan haaraan main
    git pull Hae ja yhdistä etärepon muutokset
    git push Lähetä muutokset etärepoon
    git push -u origin dev Työnnä dev ja aseta seuranta
    git merge dev Yhdistä dev nykyiseen haaraan
    git branch Näytä kaikki haarat
    git diff Näytä erot muutosten välillä
    git restore <tiedosto> Palauta tiedosto viime commitin tilaan
    git reset HEAD~ Peru viimeisin commit (muutokset jäävät)
